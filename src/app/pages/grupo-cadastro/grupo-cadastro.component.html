Cadastro de Grupo Relatório

Utilize este formulário para salvar a configuração de um novo grupo relatório. Você pode especificar um título, o nome do relatório e a consulta SQL necessária. Essas informações são importantes para gerar relatórios dinâmicos no DevExpress.

Salvar um cadastro de grupo relatório permite a geração de documentos e análises personalizadas com o DevExpress.
